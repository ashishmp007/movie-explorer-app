# 🎬 Movie Explorer App

A React based web application to explore trending movies using TMDB API.

## 🚀 Live Demo
https://movie-explorerwebapp.netlify.app/

## 🛠 Tech Stack
- React
- JavaScript
- TMDB API
- CSS
- Netlify

## ✨ Features
- Browse trending movies
- Search movies
- View movie details
- Add / Remove favorites
- Pagination
- Responsive design
